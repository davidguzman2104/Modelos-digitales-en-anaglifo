# 🎮 Visor 3D Anaglifo con Three.js y Mixamo

## 📌 Descripción del proyecto
Este proyecto consiste en el desarrollo de una aplicación web interactiva que permite visualizar modelos 3D animados provenientes de Mixamo, utilizando la librería Three.js e implementando el efecto anaglifo (3D con lentes rojo/cian).

El objetivo principal es demostrar la integración de gráficos 3D en la web y la correcta implementación de técnicas de visualización estereoscópica.

---

## 🚀 Tecnologías utilizadas

- 🌐 HTML5
- 🎨 CSS3 (Bootstrap + estilos personalizados)
- ⚙️ JavaScript (ES6 Modules)
- 🧊 Three.js
- 🎭 Mixamo (modelos y animaciones 3D)

---

## 🧠 Funcionalidades

- ✔ Carga de modelos 3D en formato `.glb`
- ✔ Visualización con efecto anaglifo (rojo/cian)
- ✔ Controles interactivos (rotación, zoom y desplazamiento)
- ✔ Interfaz moderna y responsiva
- ✔ Integración con Three.js usando módulos ES6

---

## 👓 Requisitos

Para una correcta visualización del efecto 3D:

- Uso de lentes anaglifo (rojo/cian)
- Navegador moderno (Chrome, Edge, Firefox)
- Servidor local (no abrir directamente el HTML)
